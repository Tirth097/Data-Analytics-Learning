# Cars Dataset EDA

## Objective
Perform Exploratory Data Analysis (EDA) on a Cars Dataset.

## Topics Covered
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Outlier Detection using IQR
- Histograms
- Boxplots
- Countplots
- Scatterplots
- Correlation Heatmaps

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Files
- Cars_EDA.ipynb
- cars.csv
